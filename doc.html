SAFEORA API

// REST · TON Blockchain · Instant Transfers

v1.0
Core Endpoint
POST /ton/send

Transfer TON to any wallet address instantly. Requires a valid API key passed via request headers.

Authentication & Headers
Request Headers
x-api-key:      YOUR_API_KEY
Content-Type:  application/json
Request Body
JSON Payload
{
  "wallet": "UQXXXX...",
  "amount": "0.05"
}
SAFEORA Code Example
SAFEORA Script
HTTP.post({

  url:
    "https://tbl-earn-production.up.railway.app/ton/send",

  headers: {
    "x-api-key":      "API_KEY",
    "Content-Type":  "application/json"
  },

  body: {
    wallet:  "UQXXXX...",
    amount:  "0.05"
  },

  success: "/done",
  error:   "/failed"

})
Responses
200 Success
{
  "success":  true,
  "tx_hash":  "TON_TX..."
}
4xx Error
{
  "success":  false,
  "error":    "Insufficient..."
}
